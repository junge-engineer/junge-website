# Junge 机械电气项目仓库 / Junge Mechatronics Portfolio

一个面向机械电气专业学生的静态单页展示网站，采用简洁现代的 GitHub 风格设计。适合作为个人作品集、项目展示页或校园简历展示站点。

A static one-page portfolio site for mechatronics students, styled with a clean, modern GitHub-inspired design. Suitable for personal portfolios, project showcases, or campus resume pages.

## 🚀 项目简介 / Project Overview

本仓库包含一个纯静态网站，展示以下内容：

- 项目介绍与亮点
- 技能与工具标签
- 项目卡片式展示
- 响应式移动端布局

This repository contains a static website that showcases:

- Project introduction and highlights
- Skill and tool tags
- Project card layout
- Responsive mobile-friendly design

## 📁 目录结构 / Directory Structure

- `docs/` — GitHub Pages 发布目录 / GitHub Pages publishing folder
  - `index.html` — 主页结构与内容 / main page structure and content
  - `styles.css` — 页面样式与响应式布局 / page styles and responsive layout
  - `script.js` — 简单导航交互与锚点滚动 / simple navigation and anchor scrolling
- `README.md` — 项目说明文档 / project documentation

## ✨ 主要功能 / Key Features

- 顶部导航栏：快速跳转页面章节 / Top navigation bar for fast section access
- 首页横幅：个人简介与 CTA 按钮 / Home banner with personal intro and CTA buttons
- 项目卡片：展示项目名称、标签、时间、角色、技能栈 / Project cards with name, tags, duration, role, and tech stack
- 技能模块：展示专业技能、软件工具、综合能力 / Skills section for professional abilities and tools
- 移动端适配：折叠导航与卡片式布局 / Mobile-friendly design with responsive cards
