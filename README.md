# Junge 机械电气项目仓库

一个面向机械电气专业学生的静态单页展示网站，采用 GitHub 风格的简洁现代设计。页面着重项目仓库式展示、技术栈标签、技能进度与卡片式模块。

## 文件说明

- `index.html` — 主页面结构与内容
- `styles.css` — 页面样式与响应式布局
- `script.js` — 简单导航交互与锚点滚动

## 功能亮点

- 顶部导航：Home / About / Skills / Projects / Highlights / Contact
- 首页大横幅：个人简介 + CTA 按钮
- 项目卡片：项目名称、标签、时间、角色、技术栈
- 技能模块：专业技能、软件工具、综合能力
- 适配移动端：折叠导航与卡片式布局

## 运行方式

1. 在浏览器中打开 `index.html`
2. 或者使用任意静态文件服务器查看页面
